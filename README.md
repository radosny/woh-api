# When-on-holiday REST API


### Project setup
---
Please be sure that you have mongoDB and node 6.10.3.
Setup for OSX:
- Install mongodb
```
brew install mongodb
```
- Create database with name provided in config
```
mongo
>use woh_db
```
- Switch to specific node version
```
nvm use
```
- Install all dependencies
```
npm install
```
